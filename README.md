# Asteroids Clone

Classic Asteroids game written in C# using Raylib (`Raylib-cs`).

![Asteroids Gameplay Screenshot](https://github.com/user-attachments/assets/cb97f7c2-b029-4ff9-8574-2ccc3182af0b)

## Features
* **Clean Architecture:** Scalable and well-organized C# codebase.
* **Scene Management:** Includes Title, Gameplay, and Game Over states.
* **Infinite Gameplay:** A dynamic game loop with endless waves of asteroids.
* **Polished Aesthetics:** Retro vector-style graphics and sound effects.

## Controls
* W - Thrust forward
* Mouse - Aim
* Left Click - Shoot

## Dependencies
* [**Raylib-cs**](https://github.com/raylib-cs/raylib-cs)
